##  📝 프로젝트 소개

> Palette를 통해 내가 마주하지 않았던 감정들을 물에 녹여 마주하자❗



**Palette**는 바쁜 현대인이 현재 자신이 주로 느끼는 감정을 마주하고 감정에 따라 정신적, 물질적 요소를 처방하는 플랫폼 입니다.

쉼 없이 달려가는 바쁜 일상 속, 지친 여러분들의 감정은 어떤가요? 바쁘다는 핑계로 무심히 지나치곤 했던 사소한 감정들, 하나하나 다시 깨워서 물에 녹여보는 건 어떨까요?

**Palette**는 몇가지 질문을 통해 사용자의 카메라를 이용하여 표정과 음성정보를 수집하고, 수집한 정보를 AI로 분석하여 사용자의 현재 감정상태를 파악하도록 구현하였습니다. 감정에 따른 결과를 색상으로 나타내어 주고 그에 알맞은 노래나 글귀등을 추천하고 통계를 제공합니다. 그리고 생성되는 QR코드를 실물 자판기에 인식시키면 라즈베리파이와 백엔드가 통신하여 검수 후 감정결과에 따른 물질적 요소를 제공하도록 구현하였습니다.
